# Predictive Forecasting of Care Load & Placement Demand

## Objective
Forecast Children in HHS Care using time-series and machine learning techniques.

## Models Used
- Naive Forecast
- Moving Average
- ARIMA
- SARIMA
- Exponential Smoothing
- Random Forest
- Gradient Boosting

## Evaluation Metrics
- MAE
- RMSE
- MAPE

## Dataset
UAC Program Dataset
